Building a Predictive Text Model

  • Create an algorithm for predicting the next word given a user text input using NLP

  • A large corpus of blog, news and twitter data was cleaned and analyzed

  • N-grams were extracted from the corpus and then used for building the predictive model

Methodology

  • Frequencies of n-gram vectors are computed, sorted into the bi-, tri- and quad-gram matrices

  • Dataset was cleaned by conversion to lower-cased, removed links, punctuations, extra whitespaces and special characters

  • Matrices extracted using RWeka

Shiny App interface

  • Provides a text input box for user to type a word/phrase

  • Detects words typed and predicts the next word reactively

  • Predicts using the longest, most frequent, matching N-gram
